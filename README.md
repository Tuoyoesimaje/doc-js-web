# Premium Laundry Landing Page

A professional, responsive landing page for a premium laundry and dry cleaning service in Warri, Delta State, Nigeria.

## Features

- **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
- **Modern Design** - Clean, professional aesthetic with smooth animations
- **Fast Loading** - Pure HTML/CSS/JS with no frameworks
- **Easy to Customize** - Well-organized code with CSS variables for colors
- **SEO Friendly** - Semantic HTML structure
- **Accessible** - ARIA labels and keyboard navigation support

## Sections

1. **Hero** - Eye-catching introduction with CTA buttons
2. **How It Works** - 3-step process explanation
3. **Services** - Overview of all services offered
4. **Pricing** - 3 subscription tiers with features
5. **Why Choose Us** - Trust-building feature highlights
6. **Locations** - 3 shop locations in Warri
7. **App Download** - Mobile app promotion
8. **Footer** - Contact info and quick links

## Customization Guide

### 1. Update Brand Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --color-primary: #007AFF;        /* Your main brand color */
    --color-primary-dark: #0051D5;   /* Darker shade */
    --color-primary-light: #4DA2FF;  /* Lighter shade */
}
```

### 2. Replace Images

The page uses placeholder images from Unsplash. Replace them with your own:

- Hero image (right side)
- Professional washing process
- Mobile app mockup
- Delivery service photo

### 3. Update Contact Information

In `index.html`, search for and replace:
- Phone numbers: `+234-XXX-XXX-XXXX`
- Email: `info@premiumlaundry.com`
- Addresses: `[Street Address]`

### 4. Add Real App Store Links

Update the app download buttons with your actual app store URLs.

### 5. Connect Google Maps

Add your Google Maps embed code or links to the location cards.
